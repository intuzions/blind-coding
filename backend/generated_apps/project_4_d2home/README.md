## Getting Started

### Prerequisites
- Node.js 16+
- Python 3.9+
- Docker (optional)

### Running with Docker

```bash
docker-compose up
```

### Running Manually

#### Frontend
```bash
cd frontend
npm install
npm start
```

#### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

## Access

- Frontend: http://localhost:3000
- Backend API: http://localhost:8000
- API Docs: http://localhost:8000/docs
